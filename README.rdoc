# Wikipages Project

Ruby on Rails project completed by Randy Leighton and Jennifer McCarthy. This project starts out creating one model for Contact with 7 RESTful actions. The second model is then added in for Phones, including 5 restful actions and relationship management between Contact.

- This project uses the bundle gem to manage dependencies.
- This was completed on ruby v2.0, rails 4.1.5
- Database utilized: PostgreSQL 
- Database names: wikipages_development, wikipages_test
- Testing utilizes rails helper and rspec 3.0
